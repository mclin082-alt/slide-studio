Edit loopChat edits, annotation workflow, undo, and version history.
DeliverablesFullscreen presentation, HTML download, PDF export, and persistent projects.

Core User Flow

The demo account opens directly into a portfolio-ready workspace with sample projects, so reviewers can evaluate the experience without setup.

1. Log inUse the demo account shown on the login screen.
2. Open historyReview seeded sample projects and saved versions.
3. EditAsk the AI to revise a slide or use annotations for targeted changes.
4. PresentExpand fullscreen, navigate with the keyboard, and export HTML/PDF.

Implementation Choices

The MVP uses SQLite plus local file storage for a deployable single-service architecture. The same product boundaries can move to Postgres/Supabase when real multi-user scale matters.

Auth and persistence

Email/password auth, sessions, model configs, decks, messages, comments, versions, and template selections.

Local-first storage

Generated HTML files live on disk; structured metadata lives in SQLite. This keeps the demo simple and inspectable.

Deployment-ready

Docker, Render config, environment variables, demo seeding, README, case study, and demo script are included.
